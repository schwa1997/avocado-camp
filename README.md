Landing page for an English oral training camp built with [Next.js](https://nextjs.org) and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Edit the landing page in `src/app/page.tsx` and the site metadata in `src/app/layout.tsx`.

Update the contact email in `src/app/page.tsx` (`site.contactEmail`).
